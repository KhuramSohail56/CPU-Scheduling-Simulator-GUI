Project Title: CPU Scheduling Simulator (Win32 GUI).

Description: Ke yeh simulator FCFS, SJF, aur SRTF algorithms ko visually analyze karne ke liye banaya gaya hai.

How to Run: Kyunke yeh windows.h use kar raha hai, toh likhein ke ise GCC (MinGW) ya Visual Studio ke zariye compile kiya ja sakta hai.

Compilation Command:

Bash
gcc scheduler_gui.c -o scheduler -lgdi32 -lcomctl32
